# Postman Collection Fixes

## 🎯 **Problem Summary**
The 422 Unprocessable Entity error on `POST /institution-payment-attempts/minimal` was caused by:
1. **Wrong collection variable names** - Using placeholder names instead of proper ones
2. **Missing bill generation workflow** - No way to get actual bill IDs after generation
3. **Hardcoded currency codes** - Should be pulled from credit currency ID

## ✅ **Fixes Applied**

### 1. **Collection Variable Names Updated**
**Before:**
```json
{
  "institution_entity_id": "{{institutionEntityId}}",
  "bank_account_id": "{{bankAccountId}}",
  "institution_bill_id": "{{institutionBillId}}",
  "credit_currency_id": "{{creditCurrencyId}}",
  "amount": 200.00,
  "currency_code": "USD"
}
```

**After:**
```json
{
  "institution_entity_id": "{{institutionEntityId}}",
  "bank_account_id": "{{entityBankAccountId}}",
  "institution_bill_id": "{{entityBillId}}",
  "credit_currency_id": "{{planCreditCurrencyId}}",
  "amount": 200.00
}
```

### 2. **Collection Variables Required**
```javascript
// Required collection variables
baseUrl: http://localhost:8000
authToken: {{authToken}}
institutionEntityId: {{institutionEntityId}}
entityBankAccountId: {{entityBankAccountId}}
entityBillId: {{entityBillId}}
planCreditCurrencyId: {{planCreditCurrencyId}}
paymentAttemptId: {{paymentAttemptId}}
```

### 3. **Bill Generation Workflow Added**

#### **Step 1: Generate Daily Bills**
```bash
POST {{baseUrl}}/institution-bills/generate-daily-bills?bill_date=2025-08-16
```

**Post-request Script:**
```javascript
if (pm.response.code === 200) {
    const response = pm.response.json();
    pm.collectionVariables.set("billDate", response.date);
    console.log("Bill date stored:", response.date);
}
```

#### **Step 2: Get Bill IDs (Automated)**
```bash
GET {{baseUrl}}/institution-bills/?institution_id={{institutionEntityId}}&start_date={{billDate}}&end_date={{billDate}}
```

**Post-request Script:**
```javascript
if (pm.response.code === 200) {
    const response = pm.response.json();
    if (response.length > 0) {
        const firstBill = response[0];
        pm.collectionVariables.set("entityBillId", firstBill.institution_bill_id);
        console.log("✅ Bill ID set:", firstBill.institution_bill_id);
    }
}
```

### 4. **Currency Code Handling**
- **Removed** `currency_code` from request body
- **System automatically pulls** currency code from `credit_currency_id`
- **No manual currency specification** needed
